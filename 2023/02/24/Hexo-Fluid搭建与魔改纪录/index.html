Hexo-Fluid搭建与魔改纪录

前言

去年腾讯云服务器过期了，想着采用Hexo + GitHub Pages搭建一个新的博客，主题使用Fluid。

今年~~~~感觉以前搭建的未免有点过于花哨~~~~又想折腾了，于是使用Cactus重新搭建了一遍。

今年又改回来了\(￣︶￣*))

这里记录一下自己去年使用Fluid搭建与魔改（照搬他人）blog的步骤，日后查找起来也方便。

搭建blog

准备工作

(1)需要有一个GitHub账号，没有可以去官网注册一个

(2)电脑安装Git，安装及使用教程参考Git教程

(3)由于Hexo是基于Node.js编写的，故电脑还需要安装Node.js，可以去官网进行下载

进入GitHub官网,创建一个新的代码仓库，仓库名格式必须为<用户名>.github.io。
